CSS туралы қысқаша теория

CSS (Cascading Style Sheets) – веб-беттердің стилін (түсі, қаріпі, орналасуы және т.б.) анықтайтын тіл.

CSS негізгі мүмкіндіктері

  • Стильдерді бөлек файлда сақтау – HTML кодынан бөлек сақталады (.css файлы).
  • Каскадтық ережелер – стильдер мұрагерлікпен беріледі, маңызды стильдер жоғары басымдыққа ие болады.
  • Қайта қолдану – бір CSS файлын бірнеше бетке қолдануға болады.

CSS жазу әдістері

Құжат ішінде (Inline CSS):

<p style="color: blue;">Бұл көк мәтін.</p>

Ішкі стильдер (Internal CSS):

<style>
					   p { color: blue; }
					</style>

Сыртқы файл (External CSS):

<link rel="stylesheet" href="style.css">

Негізгі CSS селекторлары

  • Элемент селекторы: p { color: red; } – барлық <p> элементтеріне әсер етеді.
  • Класс селекторы: .my-class { font-size: 20px; } – тек class="my-class" элементтеріне әсер етеді.
  • ID селекторы: #my-id { background-color: yellow; } – тек id="my-id" элементіне әсер етеді.
  • Топтық селекторлар: h1, h2, p { margin: 10px; } – бірнеше элементке бірдей стиль қолдану.
  • Ұя салынған селекторлар: div p { color: green; } – тек <div> ішіндегі <p> элементтеріне әсер етеді.
