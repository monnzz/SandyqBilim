HTML деген не?

HTML (HyperText Markup Language) – бұл веб-беттерді жасау үшін қолданылатын белгілеу тілі. Ол құжаттың құрылымын анықтайтын арнайы тэгтерден тұрады. Браузерлер HTML кодын оқып, оны қолданушыға графикалық түрде көрсетеді.

HTML құжатының негізгі құрылымы

Кез келген HTML құжаты төмендегі негізгі элементтерден тұрады:

<!DOCTYPE html>
<html>
	<head>
		<title>Бет атауы</title>
	</head>
	<body>
		<h1>Бұл негізгі тақырып</h1>
		<p>Бұл қарапайым параграф.</p>
	</body>
</html>

<!DOCTYPE html> - құжаттың HTML5 нұсқасын қолданатынын көрсетеді.

<html> - құжаттың негізгі контейнері.

<head> - метаақпараттар мен бет атауын қамтиды.

<body> - браузерде көрсетілетін негізгі мазмұнды қамтиды.

HTML тэгтері және олардың атрибуттары

HTML тэгтері – бұл элементтерді анықтайтын арнайы белгілер. Тэгтер жұппен (ашу және жабу) немесе жалғыз болуы мүмкін. Мысалы:

  • <p> - параграф жасау үшін қолданылады.
  • <a href="https://example.com"> - сілтеме жасау үшін қолданылады.
  • <img src="image.jpg" alt="Сурет"> - сурет қою үшін қолданылады.

Тэгтерде қосымша ақпарат беретін атрибуттар болуы мүмкін. Мысалы:

<a href="https://example.com" target="_blank">Сайтқа өту</a>

Бұл код сілтемені жаңа қойыншада ашуға мүмкіндік береді.

HTML-дегі негізгі элементтер

Веб-бет құру үшін келесі негізгі элементтер жиі қолданылады:

  • <h1> - <h6> - тақырыптар.
  • <p> - абзацтар.
  • <ul> <li> - тізімдер.
  • <table> - кестелер.
  • <form> - пайдаланушыдан ақпарат алу үшін формалар.

Қорытынды

HTML – веб-әзірлеудің негізгі тілі. Ол веб-беттердің құрылымын анықтайды және браузердің ақпаратты қалай көрсету керектігін белгілейді. HTML-ді жақсы меңгеру веб-әзірлеу әлеміне алғашқы қадам болып табылады.
